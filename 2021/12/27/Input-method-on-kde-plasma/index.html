Input method on kde plasma

Here I make a note to reminder me how to set up my input method on desktop(kde plams).
Recently I wiped my manjaro Linux becasue there was a problem that I could not handle.
And maybe I’ll post the probelm on another topic called “error: you need to load the kernel first”.

Ok, let us start this one to setting your imput method!!!
prerequisites:
OS: Arch Linux x86_64
DE: Plasma 5.23.4
fonts:
install fonts form AUR

git clone fonts from AUR
then cd to the folder
then type the above command to build package
next use pacman to install this package
then install will be excuted.

fonts from AUR
https://aur.archlinux.org/packages/ttf-ms-fonts/

install input framework
https://wiki.archlinux.org/title/IBus#Other_desktop_environments

under user(not root) to prevent kde plasma problem

Here ibus-chewing is for me to use traditional chinese.

Next we have setting autostart.
That is when you logining your desktop and will run ibus-daemon as your input method.

Search autostart.
Then “Add IBus Preferences”.
Next go the “Application”.
Add “ibus-daemon -drx” at the “Command”.
Then save and exit.
And logout or reboot to check this ibus panel will pop out when enter your desktop environment.
